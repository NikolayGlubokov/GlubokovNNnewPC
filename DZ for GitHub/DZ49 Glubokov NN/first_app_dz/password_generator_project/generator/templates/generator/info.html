Данное приложение предназначено для формирования пароля вместо пользователя.

Количество символов

При нажатии на стрелку данный пункт поможет выбрать необходимую длину пароля. Ваш пароль будет состоять из того числа символов, которое вы указали из выпавшего меню. Все символы будут выбираться компьютером случайным образом. При выборе пароля формируется базовый набор символов, который состоит из латинских букв малого регистра. Все символы соответствуют кодировке таблицы ASCII

Верхний регистр

При установлении галочки в данном пункте в базовый набор символов добавляются латинские буквы Верхнего регистра.

Цифры

При установлении галочки в данном пункте генератора паролей генератор случайных символов добавит цифры в базовый каталог

Спецсимволы

При установлении галочки в данном пункте генератора паролей генератор случайных символов добавит специальные символы из стандартной таблицы ASCII с №33 по №47

При нажатии на данную кнопку Вы будете перенаправлены на страницу, отображающую Ваш пароль исходя из Вашего выбора
